本版基於專案資料夾標準 v1501 修正。

重點：
1. 定時模式送法改依原廠規範：
   - 正確：send42Write16(0x0013, 0x0702)
   - 不再使用：send42Write16Pair(0x0013, 2, 7)

2. 第一~第三段時間同步寫入修正：
   - timer1: addr 7，送兩個 word，資料為 sh,sm,eh,em
   - timer2: addr 11
   - timer3: addr 15

3. schedule_page 改為整合版：
   - 殺菌排程
   - 第一~第三段定時
   - 第四~第八段附加定時
   - 第1~3段：啟用/開始/結束以設備端為主；星期以 D1 儲存值為主

4. 2026-03-31 排程頁 UI 調整：
   - 整體版型改為接近遙控器頁的單主卡風格
   - 頂部改為主卡標題、機板時間、時間校正與儲存排程按鈕
   - 不再保留上方資訊列感，避免看起來像顯示網址區塊

5. 2026-03-31 排程頁第二次 UI 調整：
   - 移除設備名稱、筆圖示與連線狀態字樣
   - 中間改為大尺寸「定時加熱」啟動/關閉按鈕，直接切換定時模式
   - 殺菌模式放在時段1前面，並與時段1~3 一起改成勾叉控制啟用/關閉
   - 排程頁取消星期一到星期日的畫面與設定，僅保留時段與開始/結束時間
   - 從 PWA 殼內仍可直接切換進入此頁
